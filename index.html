Sobre Mim

Sou estudante de Desenvolvimento de Sistemas no SENAI, com conclusão prevista para o final do ano, e futuro aluno de Sistemas de Informação. Tenho paixão por criar soluções tecnológicas que realmente fazem a diferença, sempre valorizando os pequenos detalhes que tornam cada projeto único.

Gosto de desafios, busco constantemente aprender novas linguagens e ferramentas, e acredito que a evolução acontece quando saímos da zona de conforto. Minha maior motivação é construir um futuro melhor para minha família, e por isso coloco dedicação máxima em tudo o que faço.

Recentemente, apresentei um projeto semestral em Libras, experiência que reforçou minha visão de que a tecnologia é uma poderosa ferramenta de inclusão.

Neste portfólio, você encontrará alguns dos meus trabalhos, que representam minha jornada de aprendizado e meu compromisso em crescer como desenvolvedor.

Meus Projetos

Galeria Online

Uma API RESTful para gerenciamento de imagens, construída com ASP.NET Core e Entity Framework. Ela permite o upload, listagem, atualização e exclusão de imagens, com todas as funcionalidades acessíveis.

FindCep

Desenvolvi uma aplicação que utiliza uma API para buscar dados de endereços a partir de um CEP. É simples e prático: basta digitar o número e pronto, você tem acesso a informações como rua, bairro e cidade. Um jeito rápido de encontrar o que você precisa!

Calculadora

Desenvolvi uma calculadora simples e intuitiva usando HTML, CSS e JavaScript puro. Este projeto oferece as operações matemáticas básicas (adição, subtração, multiplicação e divisão) com uma interface limpa e fácil de usar. Perfeita para cálculos rápidos no navegador!

Quiz de Personalidade Simples

Desenvolvi um Quiz de Personalidade interativo utilizando HTML, CSS e JavaScript puro. O sistema mapeia as escolhas do usuário em categorias (Guerreiro, Mago, Ladrão) através de um algoritmo de contagem de frequência, exibindo um resultado final personalizado. O projeto foca em uma interface limpa, responsiva e utiliza JavaScript nativo para toda a lógica de dados.
